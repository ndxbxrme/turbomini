# Form Validation

## What it demonstrates
- Handling form submissions in `postLoad()`.
- Inline validation with template-driven error display.
- Minimal state updates via `app.invalidate()`.

## Run it
```bash
npm run dev
```
Open `http://localhost:4173/` in your browser.
Note: the docs site hosts examples under `/turbomini/examples/<name>/`, but local dev serves from the root.

## Test it
```bash
npm test
```
