# Real App Dashboard

## What it demonstrates
- A multi-section dashboard built with TurboMini templates.
- Filtered data views with `app.invalidate()`.
- Reusable stats cards + table layout.

## Run it
```bash
npm run dev
```
Open `http://localhost:4173/` in your browser.
Note: the docs site hosts examples under `/turbomini/examples/<name>/`, but local dev serves from the root.

## Test it
```bash
npm test
```
