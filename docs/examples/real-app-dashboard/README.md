# Real App Dashboard

## What it demonstrates
- A multi-section dashboard built with TurboMini templates.
- Filtered data views with `app.invalidate()`.
- Reusable stats cards + table layout.

## Run it
```bash
npm run dev
```
Open `http://localhost:4173/real-app-dashboard/` in your browser.

## Test it
```bash
npm test
```
