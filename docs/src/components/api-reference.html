API reference

Runtime API surface, generated from the core JSDoc types.

TurboMini(basePath)

Create a TurboMini app. Use "/" for history routing or "#" for hash routing.

const app = TurboMini('/');
const hashApp = TurboMini('#');

app.template(name, text)

Register and compile a template string.

app.template('home', '<h1>Hello {{name}}</h1>');

app.$t(name, data, opts?)

Render a registered template to a string.

const html = app.$t('home', { name: 'TurboMini' });

app.controller(name, fn)

Register a route controller.

app.controller('home', () => ({ name: 'TurboMini' }));

app.defineComponent(name, def)

Register a template + optional controller as a component.

app.defineComponent('card', {
  template: '<article>{{title}}</article>',
  controller: () => ({ title: 'Card' })
});

app.start()

Start routing and render the initial route.

await app.start();

app.goto(route)

Navigate to a new route and re-render.

app.goto('/profile/42');

app.refresh() / app.refreshNow()

Schedule or immediately perform a render.

app.invalidate(); // alias of scheduled refresh
app.refreshNow();

app.setRenderStrategy(options)

Control render scheduling with microtask, raf, throttle, debounce, or idle.

app.setRenderStrategy({ mode: 'throttle', interval: 32, leading: true });

app.context

Current routing context: page, params, and data.

const { page, params, data } = app.context;

app.fetchTemplates(names, path?)

Fetch templates from the network and register them.

await app.fetchTemplates(['header', 'footer'], '/components/');

Helpers

Register or override helpers used in templates.

app.registerHelper('upper', (value) => String(value).toUpperCase());
app.unregisterHelper('upper');
const helpers = app.listHelpers();

Events

Attach global listeners or element listeners via helpers.

app.on('resize', () => {});
app.off('resize', handler);
const dispose = app.listen(button, 'click', () => {});
