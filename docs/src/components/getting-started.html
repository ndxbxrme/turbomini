Getting started

Ship a TurboMini app in minutes with the CLI or a single HTML file.

1. Install + scaffold

Use the CLI to scaffold a project with the runtime and starter files.

npm install
npx turbomini init my-app
cd my-app
npm install
npm run dev

2. Minimal HTML (no build step)

Prefer zero tooling? Drop TurboMini into a single HTML file.

<!doctype html>
<html lang="en">
  <body>
    <page></page>
    <script type="module">
      import { TurboMini } from './turbomini.js';
      const app = TurboMini('/');
      app.template('home', '<h1>Hello {{name}}</h1>');
      app.controller('home', () => ({ name: 'TurboMini' }));
      app.start();
    </script>
  </body>
</html>

3. Add theme tokens

Pull in the base theme tokens and utilities, then switch themes later.

npx turbomini theme init

The CLI will add @import "./styles/turbomini/theme.css"; to src/app.css when it finds that file. Use --no-apply to skip.

4. Run an example

Every example in this repo is runnable with a single command.

cd examples/hello-routing
npm run dev
