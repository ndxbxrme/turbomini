CLI walkthrough

Scaffold a project, enable light/dark themes, and install a web component.

1. Initialize a project

Scaffold a new TurboMini app with the built-in runtime.

npx turbomini init my-app
cd my-app
npm install
npm run dev

Open http://localhost:4173 to see the starter page.

2. Add light + dark theme tokens

Generate base theme tokens and import them into your stylesheet.

npx turbomini theme init

The CLI adds @import "./styles/turbomini/theme.css"; to src/app.css by default.

Then toggle themes by updating document.documentElement.dataset.theme.

const applyTheme = (theme) => {
  document.documentElement.dataset.theme = theme;
  return theme;
};

3. Add a web component

Install a component in copy mode (local files) and wire it into your app.

npx turbomini add tm-button tm-input

Import the components and handle events in postLoad().

import './components/tm-button.js';
import './components/tm-input.js';

app.template('default', `
  <main>
    <tm-input data-name>
      <span slot="label">Name</span>
    </tm-input>
    <tm-button data-save>Save</tm-button>
  </main>
`);

app.controller('default', () => ({
  postLoad() {
    document.querySelector('[data-save]')
      ?.addEventListener('tm-press', () => console.log('Saved'));
  }
}));

Want npm-based web components instead? Use --mode wc.

npx turbomini add tm-button --mode wc
