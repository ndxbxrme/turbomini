Templating

Build clear templates with partials, helpers, and safe rendering.

Escaping and raw HTML

Use escaped output by default and reach for triple braces only when needed.

app.template('safe', '<p>{{title}}</p>');
app.template('raw', '<div>{{{html}}}</div>');

Partials

Partials receive explicit arguments, so pass what you need.

app.template('card', '<article>{{title}}</article>');
app.template('grid', '{{#each cards as card}}{{> card title=card.title}}{{/each}}');

From the layouts example

Wrap content with a layout partial and pass explicit args.

app.template('layout',
  '<div class=\"layout\">{{> header title=title tagline=tagline}}' +
  '<section>{{{body}}}</section>{{> footer .}}</div>'
);

app.template('cards',
  '<ul class=\"cards\">{{#each cards as card}}' +
  '{{> card title=card.title owner=card.owner}}{{/each}}</ul>'
);

Helpers

Helpers are tiny formatting functions you can register globally.

app.registerHelper('upper', (value) => String(value).toUpperCase());
app.template('title', '{{upper name}}');
