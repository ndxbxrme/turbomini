Themes

TurboMini themes are driven by design tokens and CSS variables.

Base tokens

Install base tokens in src/styles/turbomini with the CLI.

npx turbomini theme init

Custom themes

Create overrides in src/styles/themes/<name> and scope with data-theme.

npx turbomini theme create brand

Light / dark switching

Toggle themes by setting document.documentElement.dataset.theme.

document.documentElement.dataset.theme = 'dark';

From the theme switching example

Persist user choice and respect system preference.

const stored = window.localStorage.getItem('tm-theme');
let theme = stored || 'system';
if (theme === 'system') {
  theme = window.matchMedia('(prefers-color-scheme: dark)').matches ? 'dark' : 'light';
}
document.documentElement.dataset.theme = theme;
