Web components

TurboMini web components emit tm-* events and accept props/attributes.

Props and attributes

Set props for state and attributes for defaults or SSR-friendly markup.

<tm-button variant="ghost" size="lg">Save</tm-button>

Events

Listen for custom events like tm-press and tm-change.

button.addEventListener('tm-press', (event) => {
  console.log(event.detail.value);
});

From the web components example

Wire component events in postLoad().

const pressButton = document.querySelector('[data-press]');
pressButton.addEventListener('tm-press', () => {
  store.pressCount += 1;
  app.invalidate();
});

SSR notes

Provide light DOM fallbacks when custom elements are not yet defined.

<tm-dialog>
  <!-- fallback content here -->
</tm-dialog>
