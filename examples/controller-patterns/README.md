# Controller Patterns

## What it demonstrates
- Composing controllers from smaller async steps.
- Shared utilities for data fetching/enrichment.
- Multiple routes wired through controller modules.

## Run it
```bash
npm run dev
```
Open `http://localhost:4173/` in your browser.
Note: the docs site hosts examples under `/turbomini/examples/<name>/`, but local dev serves from the root.

## Test it
```bash
npm test
```
