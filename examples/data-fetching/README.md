# Data Fetching

## What it demonstrates
- Loading JSON data in a controller.
- Rendering fetched lists with `{{#each}}`.
- Shared `loadItems()` helper for browser + Node tests.

## Run it
```bash
npm run dev
```
Open `http://localhost:4173/` in your browser.
Note: the docs site hosts examples under `/turbomini/examples/<name>/`, but local dev serves from the root.

## Test it
```bash
npm test
```
