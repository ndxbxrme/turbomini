# Form Validation

## What it demonstrates
- Handling form submissions in `postLoad()`.
- Inline validation with template-driven error display.
- Minimal state updates via `app.invalidate()`.

## Run it
```bash
npm run dev
```
Open `http://localhost:4173/form-validation/` in your browser.

## Test it
```bash
npm test
```
