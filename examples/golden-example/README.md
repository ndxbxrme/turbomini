# Golden Example

## What it demonstrates
- Routing across multiple pages with shared data.
- Async controller data fetched from a local JSON "API".
- Templates with `#each` and `#if` blocks.
- postLoad event wiring for custom elements.
- `tm-button`, `tm-input`, and `tm-dialog` usage.
- Theme initialization + persistence with localStorage.

## Run it
```bash
npm run dev
```
Open `http://localhost:4173/` in your browser.
Note: the docs site hosts examples under `/turbomini/examples/<name>/`, but local dev serves from the root.

## Test it
```bash
npm test
```
