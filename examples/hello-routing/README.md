# Hello Routing

## What it demonstrates
- Basic TurboMini setup with two routes (`/home` and `/about`).
- Controllers returning data into templates.
- SPA navigation via standard `<a href="/path">` links.

## Run it
```bash
npm run dev
```
Open `http://localhost:4173/` in your browser.
Note: the docs site hosts examples under `/turbomini/examples/<name>/`, but local dev serves from the root.

## Test it
```bash
npm test
```
