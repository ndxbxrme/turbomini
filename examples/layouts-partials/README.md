# Layouts & Partials

## What it demonstrates
- Layout template wrapping a body slot.
- Header/footer partials with explicit args.
- List partials for repeated UI.

## Run it
```bash
npm run dev
```
Open `http://localhost:4173/` in your browser.
Note: the docs site hosts examples under `/turbomini/examples/<name>/`, but local dev serves from the root.

## Test it
```bash
npm test
```
