# Layouts & Partials

## What it demonstrates
- Layout template wrapping a body slot.
- Header/footer partials with explicit args.
- List partials for repeated UI.

## Run it
```bash
npm run dev
```
Open `http://localhost:4173/layouts-partials/` in your browser.

## Test it
```bash
npm test
```
