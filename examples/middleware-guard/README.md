# Middleware Guard

## What it demonstrates
- Route-level guards with `app.addMiddleware()`.
- Cancelling navigation by returning `false`.
- Keeping guard state in a shared store.

## Run it
```bash
npm run dev
```
Open `http://localhost:4173/middleware-guard/` in your browser.

## Test it
```bash
npm test
```
