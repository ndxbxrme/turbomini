# State Store Pattern

## What it demonstrates
- Simple shared store object for state.
- Manual `app.refresh()` to trigger re-renders.
- Event delegation in `postLoad()`.

## Run it
```bash
npm run dev
```
Open `http://localhost:4173/state-store/` in your browser.

## Test it
```bash
npm test
```
