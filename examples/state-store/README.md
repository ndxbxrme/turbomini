# State Store Pattern

## What it demonstrates
- Simple shared store object for state.
- Manual `app.refresh()` to trigger re-renders.
- Event delegation in `postLoad()`.

## Run it
```bash
npm run dev
```
Open `http://localhost:4173/` in your browser.
Note: the docs site hosts examples under `/turbomini/examples/<name>/`, but local dev serves from the root.

## Test it
```bash
npm test
```
