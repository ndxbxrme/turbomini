# Theme Switching

## What it demonstrates
- Applying system theme preference on boot.
- Persisting manual theme changes to `localStorage`.
- Using `data-theme` on the document root.

## Run it
```bash
npm run dev
```
Open `http://localhost:4173/` in your browser.
Note: the docs site hosts examples under `/turbomini/examples/<name>/`, but local dev serves from the root.

## Test it
```bash
npm test
```
