# Theme Switching

## What it demonstrates
- Applying system theme preference on boot.
- Persisting manual theme changes to `localStorage`.
- Using `data-theme` on the document root.

## Run it
```bash
npm run dev
```
Open `http://localhost:4173/theme-switching/` in your browser.

## Test it
```bash
npm test
```
