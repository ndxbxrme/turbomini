# Web Components

## What it demonstrates
- Copy-mode web components (`tm-button`, `tm-input`, `tm-dialog`).
- `tm-*` event handling inside `postLoad()`.
- Component props and slots inside TurboMini templates.

## Run it
```bash
npm run dev
```
Open `http://localhost:4173/` in your browser.
Note: the docs site hosts examples under `/turbomini/examples/<name>/`, but local dev serves from the root.

## Test it
```bash
npm test
```
